# 🕵️‍♂️ Mafia Telegram Bot

> 🚀 Мой первый самостоятельный проект — Telegram-бот для игры в Мафию. Создан для обучения, тестирования гипотез и развития навыков Python-разработки.

##  О проекте
Это первая стабильная версия (`v1.0`). Бот уже функционален, но проект **активно развивается**. 
Изначально создавался для личного роста и расширения технических возможностей, однако база открыта для всех, кто хочет погрузиться в разработку игровых ботов или доработать механики под себя.

## 📜 Правила использования
Вы можете свободно:
✅ Запускать бота на своём сервере  
✅ Изменять код, добавлять роли, механики и настройки  
✅ Использовать как учебный материал или основу для своих проектов  

🚫 **Просьба к комьюнити:** Не выдавайте проект за свой собственный без доработок и указания авторства. Давайте развивать open-source честно! ✨
